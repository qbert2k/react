# Calendar Backend

## Run backend

Development:

```shell
npm install
npm run dev
```

Production:

```shell
npm start
```

## Start from scratch

```shell
mkdir 10-calendar-backend
cd 10-calendar-backend
npm init -y
```

### Dependencies

```shell
npm i express
npm i dotenv
npm i express-validator
npm i mongoose
npm i bcryptjs
npm i jsonwebtoken
```

Global:

```shell
sudo npm i nodemon -g
```
