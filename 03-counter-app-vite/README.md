# Dependencies

PropTypes

```shell
yarn add prop-types
```

Jest

```shell
yarn add --dev jest
yarn add -D @types/jest

yarn add --dev babel-jest @babel/core @babel/preset-env
yarn add -D whatwg-fetch 
yarn add --dev @testing-library/react
yarn add -D jest-environment-jsdom

yarn add -D @babel/preset-react
```
