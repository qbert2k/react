````shell
yarn
````

## Run PWA

Install serve (Windows)

````shell
npm install --global serve
````

Install serve (OSX / Linux)

````shell
yarn global add serve
````
Run

````shell
serve --version
serve -s build
````

## Making a Progressive Web App

````shell
npx create-react-app my-app --template cra-template-pwa
````
